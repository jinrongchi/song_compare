# song_compare

Accessed playlists using Spotify public API with Python to analyze users’ listening preference and demonstrated results on the designed website through HTML and CSS to compare users’ Spotify compatibility with others.

## Installation
Use the package manager pip to install spotipy.
```bash
pip install spotipy
```

## Usage
```bash
cd ~/Shared_Songs/Shared_Songs-main/
python main.py
```
Log in first Spotify account, wait for 5 seconds, then Log in second account. A webpage will be opened to display the list of shared songs.
